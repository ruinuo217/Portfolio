1前言

我是高婷容 Reyna，一位具備影視美學訓練、行銷企劃實戰經驗，現正專攻 Python 後端開發與 AI 系統整合的跨領域技術人才。我的背景橫跨視覺設計、電商營運與軟體工程，這樣的組合讓我能從「使用者會怎麼想」、「產品需要什麼商業價值」以及「後端工程如何穩定實現」三個維度綜合思考問題，這也是我投入軟體開發領域最大的競爭優勢。

2過往經歷與能力養成 —— 練就商業邏輯與專案落地能力

進入電商產業後，我經歷了極具挑戰的高壓工作節奏。在 Shopee 蝦皮購物任職期間，因大型檔期專案績效優異，我成功由兼職轉正，更在轉正首年獲得了 A+ 的最高績效考核（6 人小組中僅 2 位）。

在電商營運最大的收穫，不僅是具備專案管理 (PM) 能力，能橫向協調設計、BI、物流等 5 個以上跨部門團隊推進專案；更是養成了「用數據驗證決策」的習慣。我習慣請 BI 團隊撈取數據，自行產出成效分析並提出系統性流程優化提案，有效確保專案順利落地並達成預期營運指標 (KPI)。這段經驗，奠定了我在技術轉型路上，能快速釐清系統需求、精準定義問題的核心能力。

3轉職動機 —— 從前端體驗到後端架構的技術深潛

在電商追求亮眼成效的過程中，我發現自己最投入的時刻，其實是與技術團隊討論「如何調整系統邏輯以提升轉換率」。這讓我確認了自己熱愛「從無到有孕育產品」的工程建設過程。

帶著這樣的體悟，我開始系統性地補足技術能力。起初，我從接案開始，獨立完成韓式花藝品牌的 UI/UX 規劃，並協助酒商電商進行前端建置與 SEO 優化。然而，當我第一次將前端畫面與後端 API 串接，看著數據成功寫入資料庫並自動產出視覺化圖表時，我深刻體會到資料處理與後端架構在系統中的核心價值。這讓我確立了未來的方向：掌握後端架構、數據管線與 AI 應用，打造能真正解決營運痛點的底層系統，才是我期盼長期深耕的技術領域。

4學習歷程與技術深化 —— 雙軌並行的實戰開發

確立目標後，我選擇以最高強度的方式挑戰自我：白天於「AI 大數據養成班」密集鑽研後端與 AI 應用，晚上在「文化大學數位媒體學系」進修學位，長達半年以上維持每學期 20+ 學分的雙軌學習。在培訓期間與課餘時間，我將所學轉化為以下實務落地專案：

  • ■ AI 職缺能力與薪資預測系統（擔任專案組長 / ETL 與後端開發）
    • 在此專案中負責統籌開發進度與系統規格制定。從無到有規劃 ETL 資料管線，清洗並結構化破萬筆的原始資料存入關聯式資料庫。運用 Python 核心與 FastAPI 框架設計 RESTful API，並成功將後端系統與資料庫容器化，部署至 GCP 雲端環境。
  • ■ WiFi 人流分析專案（擔任專案組長 / 技術專案管理與數據架構）
    • 負責需求對齊與技術整合。目前正帶領團隊規劃人流數據的收集與分析架構（Data Pipeline），並運用 ZeroTier 建立虛擬區域網路，解決跨網域的內網穿透與團隊遠端協作需求。
  • ■ 自主開發專案：IoT 互動裝置（個人興趣 / 軟硬體整合與邊緣運算）
    • 基於對 AI 硬體整合的濃厚興趣，我利用課餘時間自主開發此專案。目前正運用 Raspberry Pi 5 實作實體互動裝置，透過 MCP 協議深度串接大語言模型 (LLM) 與 YOLO 電腦視覺模型，達成在地化的即時影像辨識。此專案目前正持續推進開發中，並以參與 COSCUP（開源人年會）發表為目標。
5未來展望

我的短期目標是發揮 Python 後端開發與系統整合的硬實力，將商業邏輯與 AI 技術 (LLM, MCP) 深度結合，為團隊打造數據驅動的智能應用與協助優化系統運作效率與資料處理的流暢度。

設計師的同理心、行銷人的商業敏感度、加上後端工程師的系統架構思維，是我在 AI 時代中獨特的綜合戰力。我期盼能將這些跨域整合能力帶入 Python 後端工程師的角色中，在系統開發的生命週期裡，既能確保程式碼的高效與穩定，又能精準擊中商業與使用者的真實需求，為企業創造具體的實質價值。
