Python 基礎語法與遠端開發

VS Code Remote SSH 環境建置、變數與資料結構、邏輯控制實作。

Part 1: 開發環境 (VS Code Remote SSH)

為什麼要用 Remote SSH？

觀念

你是一位正在學習 Python / Linux / 伺服器維運的工程師，可能會遇到：

  • 效能考量： 手邊電腦效能有限，但未來要學爬蟲、TensorFlow、LLM，需要乾淨且高效的雲端環境。
  • 部署一致性： 希望開發環境與實際部署的 Linux 主機一致，而非僅在本機跑 Python。
  • 專業流程： 學習貼近雲端的開發流程，使用 VS Code 連接 VirtualBox 中的 Ubuntu Server。
操作： 使用 VS Code 的 "Remote - SSH" 插件，連線至虛擬機並建立 Python 虛擬環境 (Virtual Environment)。

Part 2: Python 基礎語法

變數與 f-string

字串

格式化字串 (Format String) 的應用。

name = input("輸入文字")
price = input("輸入文字")
qty = input("輸入文字")

# f-string: 在字串前加 f，內部使用 {} 包裹變數
print(f"有 {name}、{price}、{qty}")

運算元綜合練習

邏輯

結合比較運算與布林運算 (Boolean)。

number = 14

# 判斷多個條件 (AND 必須全為真)
result = (number > 10) and \
         (number + 5 <= 20) and \
         (number != 15)

print(result) # 輸出: True

Part 3: 資料結構 (Data Structures)

基礎陣列 (List)

列表

有序、可變更的資料集合。

scores = [10, 20, 30]

# 透過索引值 (Index) 取值並運算
print(scores[0] + scores[1]) 
# 10 + 20 = 30

字典 (Dictionary)

鍵值對

Key-Value 配對，使用 get() 避免報錯。

student = {"name": "Alice", "age": 20}

print(student.get("name"))
# 輸出：Alice

print(student.get("grade", "未定義")) 
# 鍵不存在時，回傳預設值：未定義

元組 (Tuple) 與列表應用

複合

Tuple () 本身不可變，但若放在 List [] 中，我們可以替換 List 裡的元素。

tup_list = [
    ("To Kill a Mockingbird", "Harper Lee", 20.99),
    ("1984", "George Orwell", 15.99),
    ("The Great Gatsby", "F. Scott Fitzgerald", 10.99)
]

# 修改 List 中索引為 1 的元素 (替換整個 Tuple)
tup_list[1] = ("1984", "J.K. Rowling", 15.99)

print(tup_list)

Part 4: 邏輯控制實作

實作：比大小遊戲

練習

使用 if, elif, else 進行流程控制。

print("歡迎來到比大小遊戲！請輸入兩個數字...")

# input 預設為字串，需轉為 int 才能比較數值
num1 = int(input("請輸入第一個數字："))
num2 = int(input("請輸入第二個數字："))

if num1 > num2:
    print(f"{num1} 比 {num2} 大")
elif num2 > num1:
    print(f"{num2} 比 {num1} 大")
else:
    print(f"{num1} 和 {num2} 一樣大")

print("遊戲結束，謝謝參加！")
