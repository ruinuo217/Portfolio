Python 函數、模組與異常處理

Function 定義、Type Hinting、模組化開發與 Try-Except 除錯實戰。

Part 1: 自定義函數 (Functions)

基礎比大小函數

邏輯

使用 Type Hinting 標註參數類型。

def compare_nums(n1: int, n2: int) -> str:
    if n1 > n2:
        return f"{n1} 大 {n2}"
    elif n1 < n2:
        return f"{n1} 小 {n2}"
    else:
        return f"{n1} 等於 {n2}"

# 呼叫測試
print(compare_nums(5, 3))
x = int(input("輸入數字1: "))
y = int(input("輸入數字2: "))
print(compare_nums(x, y))

Any 類型與簡單運算

語法

從 typing 模組引入 Any，表示可接受任何型態。

from typing import Any

# 1. 接受任何型態並回傳
def identity(x: Any) -> Any:
    return x

print(identity(10))       # int
print(identity("hello"))  # str

# 2. 平方函數
def square(n: int) -> int:
    return n * n

print(square(5)) # 25

Part 2: 內建函數應用

常用工具庫實作

工具 
import random
from datetime import datetime, timedelta

# 1. 隨機數與統計
a = [random.randint(1, 100) for _ in range(5)]
print(f"列表: {a}")
print(f"總和: {sum(a)}, 最大: {max(a)}, 最小: {min(a)}")

# 2. 列表推導式 (List Comprehension)
b = [x * 2 for x in a]
print(f"加倍後: {b}")

# 3. 日期運算
today = datetime.now().date()
future = today + timedelta(days=10)
print(f"今天: {today}, 10天後: {future}")

# 4. Map 與 Split
e = input("請輸入整數列表(空格隔開): ")
f = list(map(int, e.split()))
print(f"總和: {sum(f)}")

Part 3: 模組化與日誌 (Logging)

自製模組 (Module)

結構

將功能拆分到不同檔案 (maths.py)，再由主程式 (main.py) 引用。

# maths.py
def add(a, b): return a + b
def mul(a, b): return a * b

# main.py
from maths import add, mul
print(add(2, 3)) # 5
# print(sub(10, 2)) # Error: 未引入 sub

Logging 日誌記錄

紀錄

取代 print，將執行過程寫入檔案。

import logging
logging.basicConfig(
    filename='run.log',
    filemode='w', # w: 覆寫, a: 續寫
    level=logging.INFO,
    format='%(asctime)s - %(levelname)s - %(message)s'
)

logging.info("程式開始")
result = 100 + 200
logging.info(f"計算結果: {result}")

Part 4: 異常處理 (Exception Handling)

基礎 Try-Except 結構

觀念 
try:
    # 嘗試執行的程式碼
    if a < b: print(n) # NameError
except NameError:
    print("發生錯誤: 變數未定義")
else:
    print("成功執行 (無錯誤)")
finally:
    print("無論如何都會執行 (清理資源)")

實戰：除法遊戲

練習 
def game():
    try:
        a = float(input("分子: "))
        b = float(input("分母: "))
        return a / b
    except ZeroDivisionError:
        print("錯誤：分母不得為 0")
    except ValueError:
        print("錯誤：請輸入數字")
    return None

result = game()
if result is not None:
    print(f"結果: {result}")

老師範例：狀態回傳模式

進階

使用 Tuple 同時回傳「狀態 (Boolean)」與「資料/錯誤訊息」，讓主程式更好判斷。

def calc(a, b):
    try:
        return True, a / b  # 成功
    except Exception as e:
        return False, f"錯誤: {e}"  # 失敗

# 主程式接球
status, data = calc(4, 2)

if status:
    print(f"計算成功：{data}")
else:
    print(f"發生錯誤：{data}")
