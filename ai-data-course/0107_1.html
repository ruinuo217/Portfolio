MySQL 管理與遠端連線全攻略

終端機操作、phpMyAdmin 權限管理、HeidiSQL 遠端設定與 XAMPP 整合。

Part 1: 終端機 (Terminal) 基礎操作

快速建立資料庫與使用者

指令

適合快速建立特定專案用的資料庫環境。

# 1. 以 root 身分登入 MySQL
sudo mysql -u root -p

-- 2. 建立資料庫
CREATE DATABASE my_database;

-- 3. 建立使用者 (限制 localhost 登入)
CREATE USER 'my_user'@'localhost' IDENTIFIED BY 'my_password';

-- 4. 授權 (給予該使用者對該資料庫的所有權限)
GRANT ALL PRIVILEGES ON my_database.* TO 'my_user'@'localhost';

-- 5. 刷新權限 (立即生效)
FLUSH PRIVILEGES;

-- 其他指令
show databases;  -- 查看
exit             -- 離開

Part 2: phpMyAdmin 使用者管理 (GUI)

新增使用者流程

GUI
  • Step 1: root 登入 phpMyAdmin > 點選「使用者帳號」。
  • Step 2: 點選「新增使用者帳號」。
  • Step 3: 填寫資訊 (主機名稱選 本機)。
圖：phpMyAdmin 登入資訊填寫畫面

權限設定關鍵

注意
Step 4: 設定全域權限 (Global Privileges)
請 不要勾選 (Do NOT Check) 全選！
我們不希望一般帳號擁有刪除系統資料庫的最高權限。

Step 6 後續動作：
建立完帳號後，再去「編輯權限」>「資料庫」標籤，單獨指派特定的資料庫給他。

Part 3: 開啟遠端連線 (HeidiSQL 設定)

修改 MySQL 設定檔

核心

預設 MySQL 只監聽本機 (127.0.0.1)，必須修改設定檔才能讓外部 (HeidiSQL) 連入。

圖：使用 Vim 註解掉 bind-address
# 1. 進入設定檔目錄
cd /etc/mysql/mysql.conf.d/

# 2. 編輯 mysqld.cnf
sudo vim mysqld.cnf

# 3. 找到 bind-address 並註解掉 (約第 31 行)
# 修改前： bind-address = 127.0.0.1
# 修改後： # bind-address = 127.0.0.1

# 4. 存檔離開 (:wq) 並重啟服務
sudo systemctl restart mysql.service
Step 7 驗收： 開啟 Windows 上的 HeidiSQL，輸入 Ubuntu IP 與帳密，應可成功連線。

Part 4: XAMPP 設定 (本地管遠端)

編輯 config.inc.php

進階

讓 Windows 的 XAMPP phpMyAdmin 可以切換並管理遠端的 Ubuntu 資料庫。

檔案位置： XAMPP Control Panel > Apache Config > Browse [PHPMyAdmin]
圖：在 config.inc.php 中新增遠端主機設定區塊
/* 在檔案末端新增以下遠端主機設定 */
$i++; // 索引值+1，代表這是第二台伺服器

$cfg['Servers'][$i]['auth_type'] = 'cookie';      // 認證模式
$cfg['Servers'][$i]['user'] = '';                 // 預設留空
$cfg['Servers'][$i]['password'] = '';             // 預設留空
$cfg['Servers'][$i]['extension'] = 'mysqli';      // 擴充套件
$cfg['Servers'][$i]['AllowNoPassword'] = true;    // 允許空密碼
$cfg['Lang'] = '';

// ⚠️ 重要：這裡是填寫遠端 Ubuntu 的 IP (例如 GCP/AWS 外網 IP)
$cfg['Servers'][$i]['host'] = '34.80.244.211'; 
$cfg['Servers'][$i]['connect_type'] = 'tcp';      // 連線類型
$cfg['Servers'][$i]['port'] = '3306';             // 連線埠
驗收方式： 重整本地 phpMyAdmin (http://localhost/phpmyadmin)，登入畫面會多出「伺服器選擇」下拉選單。
