數據處理核心 data_processor.py

DataProcessor 類別

核心
關鍵技巧： .copy() 資料保護 | .value_counts() 頻率統計 | .sort_index() 索引排序（時間分析關鍵） | pd.crosstab() 交叉分析 
# analysis/data_processor.py
import pandas as pd
from .schemas import COLUMN_MAPPING, OUTPUT_COLUMNS

class DataProcessor:
    def __init__(self):
        self.df = None

    def process_data(self, raw_events):
        self.df = pd.DataFrame(raw_events)
        self.df = self.df.rename(columns=COLUMN_MAPPING)

        # 時間特徵工程
        self.df['timestamp'] = pd.to_datetime(self.df['timestamp'])
        self.df['date'] = self.df['timestamp'].dt.date
        self.df['hour'] = self.df['timestamp'].dt.hour
        self.df['weekday'] = self.df['timestamp'].dt.day_name()

        self.df = self.df[OUTPUT_COLUMNS]

    def get_type_stats(self):
        return self.df['violation_type'].value_counts()

    def get_location_stats(self):
        return self.df['location'].value_counts()

    def get_hourly_stats(self):
        hours = self.df['hour'].copy()
        hour_labels = hours.apply(lambda h: f"{h:02d}-{h+1:02d}時")
        return hour_labels.value_counts().sort_index()

    def get_cross_stats(self):
        return pd.crosstab(self.df['violation_type'], self.df['location'])

報表生成 report_generator.py

ReportGenerator 類別

輸出

負責將統計結果格式化為終端機文字報表，並提供 CSV 匯出功能。

# analysis/report_generator.py
class ReportGenerator:
    def create_report(self, processor):
        print("=" * 50)
        print("工安違規事件分析報表")
        print("=" * 50)

        # 違規類型統計
        type_stats = processor.get_type_stats()
        total = type_stats.sum()
        print("\n【違規類型分析】")
        for vtype, count in type_stats.items():
            pct = count / total * 100
            print(f"  {vtype}: {count} 件 ({pct:.1f}%)")

        # 時段分析
        hourly_stats = processor.get_hourly_stats()
        print("\n【時段分析】")
        for hour, count in hourly_stats.items():
            print(f"  {hour}: {count} 件")

        # 工地違規數
        location_stats = processor.get_location_stats()
        print("\n【各工地違規數】")
        for loc, count in location_stats.items():
            print(f"  {loc}: {count} 件")

        # 交叉分析
        cross_stats = processor.get_cross_stats()
        print("\n【違規類型 x 工地 交叉分析】")
        print(cross_stats)

    def save_csv(self, processor, filename="violations.csv"):
        processor.df.to_csv(filename, index=False, encoding='utf-8-sig')
        print(f"\n已匯出 CSV: {filename}")
注意： 使用 encoding='utf-8-sig' 匯出 CSV，才能讓 Excel 正確顯示中文。一般 utf-8 在 Excel 開啟時會出現亂碼，加上 BOM 標記 (utf-8-sig) 即可解決。

入口

# main.py
from analysis.data_processor import DataProcessor
from analysis.report_generator import ReportGenerator

class ViolationAnalyzer:
    def __init__(self):
        self.processor = DataProcessor()
        self.reporter = ReportGenerator()

    def run(self, raw_events):
        # Step 1: 處理資料
        self.processor.process_data(raw_events)
        print("處理後的 DataFrame：")
        print(self.processor.df)

        # Step 2: 生成報表
        self.reporter.create_report(self.processor)

        # Step 3: 匯出 CSV
        self.reporter.save_csv(self.processor)

心得與反思

Pandas 語法邏輯

觀念

使用 Pandas 時，pd. 與 df. 的區別是初學者容易混淆的重點。

前綴 使用時機 範例
pd. 模組層級函式，用於建立或轉換資料 pd.DataFrame()、pd.to_datetime()、pd.crosstab()
df. 物件方法，對已存在的 DataFrame 操作 df.rename()、df.value_counts()、df.to_csv()
口訣： pd. = 「工廠」建立新東西；df. = 「工具」操作現有東西。

Lambda 與 Apply

觀念

理解 .apply(lambda h: f"{h:02d}-{h+1:02d}時") 這行程式碼的運作原理。

  • .apply() = 批次處理，不需要寫 for 迴圈，對 Series 中每個元素套用指定函式
  • lambda = 匿名單行函式，適合簡短的轉換邏輯
  • f"{h:02d}" = 格式化為兩位數字（例如 9 → 09）
等價的傳統寫法
# 傳統 for 迴圈寫法
result = []
for h in hours:
    label = f"{h:02d}-{h+1:02d}時"
    result.append(label)

# 使用 apply + lambda（推薦）
result = hours.apply(lambda h: f"{h:02d}-{h+1:02d}時")

開發小技巧

技巧
  • sort_index()：用於時間類資料排序。value_counts() 預設按數量排序，加上 sort_index() 可改為按索引（時間）排序，讓時段報表更直覺。
  • encoding='utf-8-sig'：匯出 CSV 時加上 BOM 標記，確保 Excel 開啟不會亂碼。這是處理中文 CSV 的必備技巧。
  • index=False：呼叫 to_csv() 時避免 Pandas 自動產生的索引欄位寫入檔案，保持資料乾淨。
  • API 分離設計：ViolationAnalyzer 作為對外介面，DataProcessor 負責實作細節。未來擴充功能時只需新增方法，不影響既有流程。
