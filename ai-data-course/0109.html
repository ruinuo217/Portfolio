Python 進階語法與資料處理

錯誤控制、進階迴圈與迭代、檔案讀寫以及 JSON 格式轉換。

Part 1: 判斷式與迴圈控制

錯誤檢查 (Assert)

除錯

確保變數符合條件，否則程式會中斷並報錯。

a = 1
b = 0

# 若 b 為 0，則跳出錯誤訊息 "不零"
assert b != 0, "不零"

print(a/b)

兩種基礎迴圈

迴圈 
# 1. For 迴圈 (累加 1 到 5)
b = 0
for a in range(1, 6):
    b += a
print(b)

# 2. While 迴圈
a = 1 # 計數器
b = 0 # 總和
while a <= 5:
    b = b + a 
    a = a + 1 
print(b)

進階迴圈 (Break & Continue)

邏輯

無限迴圈與條件跳轉的應用。

a = 0 # 計數器
b = 0 # 次數

while True:
    a += 1
    if a % 2 == 0: # 如果是偶數
        if a == 100: # 到 100 停止
            break
        continue # 跳過本次迴圈 (不執行 b+=1)
    
    b += 1 # 只有奇數時會執行到這

print(a, b)

Part 2: 迭代與資料處理

迭代器 (Iteration)

原理

使用 iter() 與 next() 手動取值。

my_list = [1, 2, 3, 4]
it = iter(my_list)

while True:
    try:
        num = next(it)
        print(num)
    except StopIteration:
        break # 取完後停止

字串切割 (Split)

應用

將輸入字串依逗號切割成列表。

a = input("輸入數字1,2").split(",")

if a[0] > a[1]:
    print(f"{a[0]} > {a[1]}")
elif a[0] < a[1]:
    print(f"{a[0]} < {a[1]}")
else:
    print(f"{a[0]} = {a[1]}")

Part 3: 檔案讀寫與 JSON

檔案讀取 (Read)

IO

讀取文字檔並印出偶數行。

f = open("0109.txt", "r")
b = 1

for content in f.readlines():
    if b % 2 == 0:
        print(content)
    b = b + 1

f.close()

JSON 轉換 (Loads/Dumps)

格式 
import json

# 1. Loads: JSON 字串 -> Python 字典
str1 = '{"id": 123, "Name": "wsrsw"}'
j = json.loads(str1)
print(type(j)) # dict

# 2. Dumps: Python 字典 -> JSON 字串
j_data = {'id': 123, 'Name': 'wsrsw'}
str2 = json.dumps(j_data)
print(type(str2)) # str

JSON 實戰練習

練習

讀取 0109.json 檔案並篩選出數學成績。

import json

# 讀取 JSON 檔案
f = open("0109.json", "r")
date = json.load(f) # 注意這裡是 load 不是 loads
f.close()

print("讀取類型", type(date))

# 遍歷資料
a = date['contents']
for item in a:
    if item['subject'] == 'Math':
        print("數學成績:", item['score'])
