CSS 樣式設計全攻略

選擇器權重、盒模型、排版系統 (Flex/Position)、BEM 命名與 CSS 動畫實戰。

Part 1: 選擇器與權重 (Selectors & Specificity)

CSS 選擇器大全

核心
1. 簡單與組合選擇器
名稱語法說明範例
後代div p選取 div 內的所有 p (不論層級)。.container p
子元素ul > li只選取第一層子元素。nav > ul
相鄰兄弟h1 + p選取緊接在 h1 後的第一個 p。label + input
一般兄弟h1 ~ p選取 h1 之後同層級的所有 p。h2 ~ p
2. 屬性選擇器 (Attribute)
語法說明
[attr]有該屬性的元素 (如 [disabled])。
[attr="val"]屬性值完全相符 (如 [type="text"])。
[attr^="val"]以 val 開頭 (如 [src^="https"])。
[attr$="val"]以 val 結尾 (如 [src$=".png"])。
3. 偽類 (Pseudo-class)

選取元素的「狀態」。

  • :hover - 滑鼠懸停
  • :active - 點擊當下
  • :focus - 獲得焦點 (表單)
  • :first-child / :last-child
  • :nth-child(n) - 第 n 個元素
  • :not(.class) - 排除特定元素

權重計算 (Specificity)

必考

當樣式衝突時，瀏覽器根據權重分數決定聽誰的。

選擇器類型分數範例
!important∞(最強制)
Inline Style1000style="..."
ID100#header
Class / 偽類10.btn, :hover
Element / 偽元素1div, ::before
口訣： ID > Class > Tag。盡量少用 !important。

Part 2: 視覺樣式與排版 (Visual & Layout)

文字與背景

樣式
文字 (Typography)
  • font-family: 字型 ("Arial", sans-serif)。
  • line-height: 行高 (如 1.5)。
  • letter-spacing: 字距。
  • text-align: 對齊 (justify 真對齊)。
背景 (Background)
.bg {
  background-image: url("img.jpg");
  background-size: cover; /* 填滿 */
  background-position: center;
  background-repeat: no-repeat;
}

Box Model 與 Display

佈局
Box Model

Content -> Padding -> Border -> Margin

box-sizing: border-box;
必設！讓 padding/border 包含在寬度內，不爆框。
Display 模式
  • block: 獨佔一行 (div, p)。
  • inline: 並排，無法設寬高 (span, a)。
  • inline-block: 並排且可設寬高。
  • flex: 彈性排版容器。
  • none: 隱藏 (不佔位) vs visibility: hidden (佔位)。

定位 (Position) 與裝飾

進階
Position 定位
static預設值，照順序排。
relative相對自己原位偏移 (佔位)。
absolute絕對定位，找非 static 的父層定位。
fixed固定視窗，不隨滾動移動 (導覽列)。
sticky滾到特定位置後黏住。
裝飾屬性
  • Border-radius: 圓角 (50% 變圓形)。
  • Box-shadow: 陰影 (x y blur spread color)。
    box-shadow: 4px 4px 10px rgba(0,0,0,0.2);
  • Overflow: 溢出處理 (hidden, scroll, auto)。

Part 3: CSS 架構 (BEM)

BEM 命名規範

規範

Block-Element-Modifier：讓 CSS 類別名稱具備結構性與可讀性，像蓋房子一樣。

1. Block (區塊)

獨立的組件，如 .menu, .card。

2. Element (元素)

區塊內的零件，用 __ 連接。
如 .menu__item。

3. Modifier (修飾符)

狀態或變體，用 -- 連接。
如 .menu__item--active。

<!-- HTML 結構範例 -->
<nav class="menu">
  <ul class="menu__list">
    <li class="menu__item menu__item--active">首頁</li>
    <li class="menu__item">關於我們</li>
  </ul>
</nav>

Part 4: 動畫與互動 (Animation)

Transition (過渡)

互動

適合簡單的 A 到 B 狀態改變 (如 Hover)。

.btn {
  transition: background 0.3s ease;
}
.btn:hover {
  background: red;
}

Transform (變形)

變形

改變形狀與位置，不影響文件流。

  • translate(x, y): 位移。
  • scale(1.2): 縮放。
  • rotate(45deg): 旋轉。
  • skew(10deg): 傾斜。

Animation & AOS

進階
Keyframe Animation (關鍵影格)

適合複雜、自動播放的動畫。

@keyframes slide {
  0% { transform: translateX(0); }
  100% { transform: translateX(100px); }
}
.box {
  animation: slide 2s infinite;
}
AOS (Animate On Scroll)

捲動視窗觸發動畫的 JS 套件。

<!-- 引入 CSS/JS 後 -->
<div data-aos="fade-up">
  當我捲動出現時，我會向上淡入
</div>

<script>
  AOS.init();
</script>
