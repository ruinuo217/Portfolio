HTML 基礎與表單設計

超文本標記語言 (HTML5)、語意化標籤 (Semantic Tags)、Resume 實作與 Form 表單應用。

0. HTML 基礎 (The Basics)

HTML 文件結構與開發環境

核心

HTML (HyperText Markup Language) 是網頁的骨架。推薦使用 VS Code 進行開發。

標準文件結構 (Boilerplate)
<!DOCTYPE html>
<html lang="zh-TW">
<head>
    <meta charset="UTF-8"> <!-- 指定編碼防止亂碼 -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0"> <!-- 手機版RWD適配 -->
    <title>我的網頁</title>
</head>
<body>
    <h1>Hello World</h1>
    <p>這是一個段落。</p>
</body>
</html>

常用文字標籤

常用
  • <h1>~<h6>: 標題 (Title)，權重 h1 最大。
  • <p>: 段落 (Paragraph)，會自動換行。
  • <a href="...">: 超連結 (Anchor)。
  • <img src="...">: 圖片 (Image)，單標籤。
  • <ul>/<ol> + <li>: 清單列表。
  • <hr>: 水平分隔線。

Block vs Inline

觀念
Block (區塊) Inline (行內)
獨佔一行 (換行) 與他人並排 (不換行)
<div>, <p>, <h1> <span>, <a>, <img>
用途： 排版、容器 用途： 局部文字樣式

1. 語意化標籤實作 (Resume Project)

為什麼要用語意化標籤？

SEO

讓機器 (搜尋引擎、螢幕閱讀器) 讀懂網頁結構，而不僅僅是使用 <div>。

主要結構標籤
  • <header>: 頁首，通常包含標題或導航。
  • <nav>: 導航選單 (Navigation)。
  • <main>: 主要內容區 (一個頁面只能有一個)。
  • <footer>: 頁尾，版權宣告或聯絡資訊。
內容區塊標籤
  • <article>: 獨立完整的內容 (如部落格文章)。
  • <section>: 主題分區 (如履歷的「學歷區」、「經歷區」)。
  • <aside>: 側邊欄或補充資訊 (與主題間接相關)。
  • <address>: 聯絡人資訊區塊。

2. HTML 表單設計 (Forms)

<form> 容器屬性

設定
  • action: 資料要送到哪裡 (後端 URL)。
  • method: 傳送方式。
    • GET: 參數在網址上 (搜尋)。
    • POST: 參數在封包內 (註冊、登入)。
  • enctype: 傳送檔案時需設為 multipart/form-data。

<input> 常用類型

元件
type="text"單行文字 (預設)
type="password"密碼 (隱藏字元)
type="radio"單選 (需同名 name)
type="checkbox"多選 / 核取方塊
type="date"日期選擇器
type="file"檔案上傳
type="submit"提交按鈕

表單範例程式碼

實作 
<form action="/submit" method="post">
    <!-- Label 的 for 屬性要對應 input 的 id，點擊文字也會聚焦 -->
    <label for="username">姓名:</label>
    <input type="text" id="username" name="user" required placeholder="請輸入姓名">

    <label>性別:</label>
    <input type="radio" name="gender" value="M"> 男
    <input type="radio" name="gender" value="F"> 女

    <label for="bio">自我介紹:</label>
    <textarea id="bio" name="bio" rows="5"></textarea>

    <label for="city">居住城市:</label>
    <select id="city" name="city">
        <option value="tp">台北</option>
        <option value="kh">高雄</option>
    </select>

    <button type="submit">送出</button>
</form>
